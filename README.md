# 🎨 3D Photo Gallery Hover Effect

This project is a modern **3D image gallery** built using pure **HTML and CSS**, featuring smooth hover interactions and depth-based animations.

## 🚀 Features

* ✨ 3D perspective gallery layout
* 🎯 Smooth hover animations
* 🌗 Dynamic brightness adjustments
* 🔄 Sibling-based interaction effects
* 💡 Pure CSS (no JavaScript used)

## 🧠 How It Works

The gallery uses:

* `transform: perspective()` to create a 3D space
* `translateZ()` for depth movement
* `rotateY()` for angled side effects
* `filter: brightness()` for focus highlighting
* Advanced selectors like `:has()` for interactive sibling behavior

When hovering on an image:

* The active image pops forward
* Nearby images rotate and shift in 3D space
* Non-focused images dim to create depth and focus

## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox, Transforms, Filters, Advanced Selectors)
